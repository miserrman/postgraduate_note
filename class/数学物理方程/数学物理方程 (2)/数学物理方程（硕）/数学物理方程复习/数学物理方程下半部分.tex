\subsection*{\wuhao\heiti  （一）\ $\delta$函数}
1. $\delta$ 函数的定义

  满足下面二式的函数
\emph{ $$ \left\{
\begin{array}{c}\delta(x-x_{0})=\left\{\begin{array}{c}0,\quad\quad x\neq x_{0}
\\*[3pt]
\infty,\quad x=x_{0}\end{array}\right.
\\*[3pt]
\int^{\infty}_{-\infty}\delta(x-x_{0})dx=1
\end{array}
\right. $$} 称为$\delta$函数.

2. $\delta$函数的性质

（1）设 $f(x)$在$(-\infty,\infty)$ 上连续，则
$$\int^{\infty}_{-\infty}f(x)\delta(x-x_{0})dx=f(x_{0})$$

(2)\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
$\delta(x_{0}-x)=\delta(x-x_{0})$ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \

即， $\delta$函数是一偶函数

(3)\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
$\delta[\varphi(x)]=\Sigma^{k}_{i=1}\frac{\delta(x-x_{i})}{|\varphi^{\prime}(x_{i})|}$

其中,$\varphi(x_{i})=0(i=1,2,\cdots,k).$

(4)若定义 $\frac{d}{dx}\delta(x)=\delta^{\prime}(x)$称为$\delta$
函数的导数，则
$$(x-x_{0})\delta^{\prime}(x-x_{0})=-\delta(x-x_{0})$$

$$\int^{\infty}_{-\infty}f(x)\delta^{(n)}(x-x_{0})dx=(-1)^{(n)}f^{(n)}(x_{0})$$
$$\delta^{\prime}(x_{0}-x)=-\delta^{\prime}(x-x_{0})$$
即 $\delta$函数的导数是以奇函数, 其中$f(x)$
是具有连续导数的任意函数.

3.  高维 $\delta$函数

n维($n\geq2$ 的整数)$\delta$ 函数可视为n个一维$\delta$ 函数的乘积,如
$$\delta(M-M_{0})=\delta(x-x_{0},y-y_{0},z-z_{0})\hspace*{75pt}$$
$$=\delta(x-x{0})\delta(y-y_{0})\delta(z-z_{0})$$
于是易于得到$$\iiint^{\infty}_{-\infty}f(M)\delta(M-M_{0})dxdydz=f(M_{0})$$
其中$$\delta(M-M_{0})=\left\{\begin{array}{c}0,\quad\quad M\neq
M_{0}
\\*[3pt]
\infty,\quad M=M_{0}\end{array}\right. $$
$$\iiint^{\infty}_{-\infty}\delta(M-M_{0})dxdydz=1$$
而$f(M)$在$(-\infty<x,y,z<\infty)$中连续.

\ \ \ \ \ (2.5.39)

\subsection*{\wuhao\heiti  （一） 泛函和泛函的极值}
1.泛函

一个变数J,其值取决于函数$y(x)$,就称之为$y(x)$的泛函,通常记为$$J[y(x)]=\int^{b}_{a}F(x,y,y^{\prime})dx$$其中,
$F(x,y,y^{\prime})$称为泛函$J[y(x)]$的核.

2. 变分

(1)若函数$y(x)$有微小的变形$t\eta(x)$(t为小参数则记
$$\delta(y)=t\eta(x)$$称为函数$y(x)$的变分.变分和微分可交换次序.

(2)若函数$y(x)$有变分$\delta y$,则称J的改变
$$\Delta J[y(x)+\delta y]-J[y(x)]\hspace*{75pt}$$
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
$=\int^{b}_{a}[\frac{\partial F}{\partial y}\delta y+\frac{\partial
F}{\partial y^{\prime}}\delta y^{\prime}+t$ 的高阶小量]$dx$

的线性主部(即略去了$t$的高阶小量的部分)为泛函$J[y(x)]$的变分,记作$$\delta
J=\int^{b}_{a}(\frac{\partial F}{\partial y}\delta y+\frac{\partial
F}{\partial y^{\prime}}\delta y^{\prime})dx$$

3.泛函的极值与变分问题

泛函的极值问题称为变分问题.研究变分问题的方法可归结为两类,一类为间接法,即把变分问题转化为解微分方程(Euler方程);
另一类为直接法,即把变分问题转化为直接求普通的多元函数的极值问题,如Ritz方法就是-直接法.

(1)间接法与Euler方程\ \ \ \ \ 泛函J有极值的必要条件为
$$\delta J=0$$
由此可推出Euler方程, 解Euler方程即求得J的极值.

$1^{\circ}$ $y(x)$的泛函的变分问题 $$\delta J=\delta
\int^{b}_{a}F(x,y,y^{\prime})=0$$的Euler方程是
$$\frac{\partial F}{\partial y}-\frac {d}{dx}\frac {\partial F}{\partial y^{\prime}}=0$$
当F不显含$x$时,(2.6.6)式化为$$y^{\prime}\frac{\partial F}{\partial
y^{\prime}}-F=C$$其中C为常数.

$2^{\circ}$n个函数$y_{1}(x)$,$y_{2}(x)$,\ldots$y_{n}(x)$的泛函的变分问题$$\delta
J=\delta \int
^{b}_{a}F(x,y_{1},y_{2},\ldots,y_{n};y_{1}^{\prime},y_{2}^{\prime},\ldots,y_{n}^{\prime})dx=0$$的Euler方程是
$$\frac{\partial F}{\partial y_{i}}-\frac {d}{dx}\frac {\partial F}{\partial
y_{i}^{\prime}}=0,i=1,2,\ldots,n$$

$3^{\circ}$
$y_{i}(x)$及其高阶导数($i=1,2,\ldots,n$)的泛函的变分问题$$\delta
J=\delta \int
^{b}_{a}F(x,y_{1},y_{2},\ldots,y_{n};y_{1}^{\prime},y_{2}^{\prime},\ldots,y_{n}^{\prime};y_{1}^{(m)},y_{2}^{(m)},\ldots,y_{n}^{(m)})dx=0$$的Euler方程是
$$\frac{\partial F}{\partial y_{i}}-\frac {d}{dx}\frac {\partial F}{\partial
y_{i}^{\prime}}+\frac {d^{2}}{dx^{2}}\frac {\partial F}{\partial
y_{i}^{\prime \prime}}+\ldots +
(-1)^{m}\frac{d^{m}}{dx^{m}}\frac{\partial F}{\partial
y^{(m)}_{i}}$$
$$=0,i=1,2,\ldots,n$$
$4^{\circ}$多元函数如$u(x,y,z)$的泛函的变分问题$$\delta J=\delta
\iiint_{v}F(x,y,;u;u_{x},u_{y},u_{z})dxdydz=0$$的Euler方程为$$\frac{\partial
F}{\partial u}-\frac{\partial}{\partial x}\frac{\partial F}{\partial
u_{x}}-\frac{\partial}{\partial y}\frac{\partial F}{\partial
u_{y}}-\frac{\partial}{\partial z}\frac{\partial F}{\partial
u_{z}}=0$$

$5^{\circ}$
$J[y(x)]=\int^{b}_{a}F(x,y,y^{\prime})dx$带有附加条件$$\int^{b}_{a}G(x,y,y^{\prime})dx=l$$
 变分问题$$\delta\int^{b}_{a}[F(x,y,y^{\prime})+\lambda
 G(x,y,y^{\prime})]dx=0$$的Euler方程是$$\frac{\partial F}{\partial y}+\lambda\frac{\partial G}
 {\partial y}-\frac{d}{dx}(\frac{\partial F}{\partial y^{\prime}}+\lambda\frac{\partial G}{\partial y^{\prime}})=0$$
其中$\lambda$为参数.

(2)直接法与Ritz方法\ \ \ \ Ritz方法的基本要点是, 不把泛函$J[y(x)]$
放在它的全部定义域内来考虑,
而把它放在其定义域的某一部分来考虑.具体做法是

$1^{\circ}$ 取某种完备函数系$\varphi_{1}(x),\varphi_{2}(x)\cdots$

$2^{\circ}$令$$y_{n}(x)=f(\varphi_{1},\varphi_{2},\cdots,\varphi_{n},C_{1},C_{2},\cdots,C_{n})$$其中,
$C_{i}(i=1,2,\cdots,n)$为待定系数,$f$和$\varphi_{i}$的形式可根据边界条件等适当选择.常选多项式或三角函数系.则$$J[y(x)]\thickapprox
J[y_{n}(x)]=\phi(C_{1},C_{2},\cdots,C_{n})$$

$3^{\circ}$令$\frac{\partial\phi}{\partial C_{i}}=0$
由此可求得$C_{i}$,从而确定近似解$y_{n}(x)$:而$$y(x)=\lim_{n\rightarrow
\infty}y_{n}(x)$$

\subsection*{\wuhao\heiti  （一）Bessel方程及柱函数}

1.Bessel方程的级数解

由常微分方程的级数解法得，Bessel方程(3.2.3)在正则奇点$x=0$点领域有两个级数解(并不一定县线性无关)
$$y_{1}(x)=\sum^{\infty}_{n=0}\frac{(-1)^{n}c_{0}\Gamma(\nu+1)}{2^{2n}n!\Gamma(\nu+n+1)}x^{2n+\nu}$$
$$y_{2}(x)=\sum^{\infty}_{n=0}\frac{(-1)^{n}c_{0}\Gamma(-\nu+1)}{2^{2n}n!\Gamma(-\nu+n+1)}x^{2n-\nu}$$
该二级数分别在$0\leq x<\infty$和$0<x<\infty$中收敛.

2.柱函数和Bessel方程的通解

(1)第一类柱函数--Bessel函数

分别取(3.2.4)和(3.2.5)式中$c_{0}=\frac{1}{2^{\nu}\Gamma(\nu+1)}$和$c_{0}=\frac{1}{2^{-\nu}\Gamma(-\nu+1)}$,
并将此时的$y_{1}(x)$和$y_{2}(x)$记为$J_{\nu}(x)$和$J_{-\nu}(x)$,分别称之为$\nu$和$-\nu$阶的{\heiti
Bessel函数}或{\heiti
第一类柱函数},则$$J_{\nu}(x)=\sum^{\infty}_{k=0}\frac{(-1)^{k}}{k!\Gamma(\nu+k+1)}(\frac{x}{2})^{2h+\nu}$$
$$J_{-\nu}(x)=\sum^{\infty}_{k=0}\frac{(-1)^{k}}{k!\Gamma(-\nu+k+1)}(\frac{x}{2})^{2h-\nu}$$它们均为Bessel方程(3.2.3)的特解.

当$\nu\neq
n(n=0,\pm1,\pm2,\cdots)$时,Bessel方程(3.2.3)的通解为$$y_{c}=C_{\nu}J_{\nu}(x)+D_{\nu}J_{-\nu}(x)$$

当$\nu=n$时,$$J_{-n}(x)=(-1)^{n}J_{n}(x)$$

(2)第二类柱函数--Neuman函数

定义$$N_{\nu}(x)=\frac{\cos \nu\pi J_{\nu}(x)-J_{-\nu}(x)}{\sin \nu
\pi}$$为{\heiti
第二类柱函数},它是Bessel方程(3.2.3)的另一特解.又叫{\heiti
Neuman函数}.而$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \
 \ \ \ \ \ \ $ $N_{n}(x)=\lim_{\nu\rightarrow n}\frac{\cos\nu\pi
J_{\nu}(x)-J_{-\nu}(x)}{\sin\nu\pi}$

$\ \ \
=\frac{2}{\pi}J_{n}(x)\ln\frac{x}{2}-\frac{1}{\pi}\sum^{n-1}_{k=0}$

$\ \ \
-\frac{1}{\pi}\sum^{\infty}_{k=0}\frac{-1}{k!(n+k)!}[\psi(k+1)+\psi(k+n+1)](\frac{x}{2})^{2k+n}$

无论$\nu=n$与否,Bessel方程(3.2.3)的通解均可表示为$$y_{c}=A_{\nu}J_{\nu}(x)+B_{\nu}N_{\nu}(x)$$

(3)第三类柱函数--Hankel函数

定义$$\left\{\begin{array}{c}H^{(1)}_{\nu}(x)=J_{\nu}(x)+iN_{\nu}(x)
\\*[3pt]
H^{(2)}_{\nu}(x)=J_{\nu}(x)-iN_{\nu}(x)
  \end{array}\right.$$为{\heiti
  第三类柱函数},又叫Hankel函数.它是Bessel函数(3.2.3)的又一形式的特解.

无论$\nu=n(n=0,\pm1,\pm2,\cdots)$与否,Bessel方程(3.2.3)的通解均可由$J_{\nu}(x),J_{\nu}(x),H^{(1)}_{\nu}(x)$和$H^{(2)}_{\nu}(x)$之中的任意两个线性组合而成.

3.Bessel方程的本征值问题

(1)本征值问题$$\left\{\begin{array}{c}\rho^{2}R^{\prime\prime}(\rho)+\rho
R^{\prime}(\rho)+(k^{2}\rho^{2}-n^{2})R=0
\\*[3pt]
R(a)=0
  \end{array}\right.$$的本征值为$$k^{n}_{m}=\frac{x^{n}_{m}}{a},m=1,2,3,\cdots$$相应的本征函数为$$R_{m}(\rho)=J_{n}(\frac{x^{n}_{m}}{a}\rho)$$
其中,$x^{n}_{m}$为$$J_{n}(x)=0$$的第$m$个正根,称之为$J_{n}(x)$的第$m$个{\heiti
零点}.

(2)本征值问题$$\left\{\begin{array}{c}\rho^{2}R^{\prime\prime}(\rho)+\rho
R^{\prime}(\rho)+(k^{2}\rho^{2}-n^{2})R=0
\\*[3pt]
\frac{dR}{d\rho}|_{\rho-a}=0
  \end{array}\right.$$的本征值为$$\widetilde{k}^{n}_{m}=\frac{\widetilde{x}^{n}_{m}}{a},m=1,2,3,\cdots$$
相应的本征值函数为$$R_{m}(\rho)=J_{n}(\frac{\widetilde{x}^{n}_{m}}{a}\rho)$$其中$\widetilde{x}^{n}_{m}$为$$J^{\prime}_{n}(x)=0$$
的第$m$个正根,称之为$J^{\prime}_{n}(x)$的第$m$个零点.
